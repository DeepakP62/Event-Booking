A RESTful API built with Node.js, Express.js, and MongoDB that allows users to sign up, log in, view events, and book tickets. This application demonstrates a backend API focused on authentication, event management, and ticket booking.

Features
 User Signup & Login (Authentication)
 Secure Password Hashing
 JWT Token Based Authorization
 View All Events
 Book Tickets for Events
 CRUD for Events (Create, Read, Update, Delete)
 MongoDB Database

Technology	Usage:
  Node.js	--> Runtime Environment
  Express.js --> API Framework
  MongoDB	--> Document Database
  Mongoose	--> ODM for MongoDB
  JWT	--> Authentication Tokens
  bcrypt	--> Password Hashing
  Postman	API --> Testing
